# Auxi — Homepage

Marketing / landing website for **Auxi** (personal-wardrobe + AI outfit recommender).

Part of the `wardrobe_project` umbrella monorepo, wired in as a git submodule
alongside `auxi` (mobile), `auxi-web`, and `wardrobe-backend`.

## Status

Scaffold pending — this is the initial seed commit that establishes the `main`
branch so the repo can be consumed as a submodule. The actual site (framework,
content, deploy target) lands in a follow-up.

## Local setup

```bash
# From the umbrella root, this repo lives at homepage/
git submodule update --init homepage
```
